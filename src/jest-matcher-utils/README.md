# jest-matcher-utils

Upstream: https://github.com/facebook/jest/tree/v28.0.0/packages/jest-matcher-utils

Helpers used by `expect`'s matcher implementations: `stringify`, `printExpected` / `printReceived` / `printWithType`, the `ensure*` argument validators, `printDiffOrStringify`, `diff`, `pluralize`, `getLabelPrinter`, and `matcherHint` for assembling assertion messages.
