# jest-mock

Upstream: https://github.com/facebook/jest/tree/v27.4.7/packages/jest-mock

Function and module mocking for Jest Roblox. Provides `ModuleMocker` which creates mock functions (`fn`), spies on object methods and Instance methods (`spyOn`), and manages mock lifecycle (`clearAllMocks`, `resetAllMocks`, `restoreAllMocks`). Also supports mocking globals via `jest-mock-genv` and data model instances via `jest-mock-rbx`.
