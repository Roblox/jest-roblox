# jest-test-result

Upstream: https://github.com/facebook/jest/tree/v28.0.0/packages/jest-test-result

Types and helpers for test results. Defines `TestResult`, `AggregatedResult`, `AssertionResult`, and related types. Provides `addResult`, `buildFailureTestResult`, `createEmptyTestResult`, and `formatTestResults` utilities used by jest-runner and jest-reporters.
